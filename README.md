A React-based CSAT (Customer Satisfaction) Campaign Builder that allows users to configure customer feedback popups and see a live mobile preview in real-time.

Live Demo : https://assignmentappvercel.netlify.app/

GitHub Repository : 
--------------------------------------------------------------------------------------------------------------------------------
Project Overview :
This project was developed as part of a Frontend Intern Assignment.

The application enables users to:
Configure feedback popup content
Customize popup styling
Manage dynamic feedback options
Upload thank-you page media
Preview changes instantly on a mobile device layout

All changes are reflected in real-time without requiring a page refresh or save button.
